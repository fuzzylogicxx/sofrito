Modal Dialogues

Credit: Scott O’Hara’s Accessible Modal Dialogs with Vanilla JS

Without Javascript, the modal content simply shows on the page. Therefore it probably makes little sense to do as I used to and use an anchor linking to a real page as the pre-enhancement baseline (although that is an option) since the modal content is already available. Better instead to use a button. I like the option of using the hidden attribute (rather than disabled) on the button so that without Javascript enhancement it doesn’t show at all.
