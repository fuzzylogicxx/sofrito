Aspect Ratio Maintenance

Usually on any given image we’ll want to ensure a specific aspect ratio – usually the original aspect ratio e.g. 16:9 or 4:3 but not always, for example where we want a square thumbnail. Often images have an HTML height attribute set inline and when this combines with our responsive CSS of img { width: 100%; } the image will look broken.

At some point in the future this may be handled automatically by the browser – see what-if-we-got-aspect-ratio-sized-images-by-doing-almost-nothing

Developers tend to wrap images in a div to handle maintaining aspect ratio. See https://css-tricks.com/aspect-ratio-boxes/ (todo: try Thierry Koblentz’s idea!)
